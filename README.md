# Основание проекта SWARM_MIL

Проект содержит исследовательскую основу для поэтапного моделирования
роевого управления беспилотными воздушными судами в `MATLAB` и
`Simulink` версии `R2025b`.

## Где читать описание

Основное описание проекта находится в [OPISANIE.md](OPISANIE.md).

## Текущее состояние

После этапа 7 в репозитории подготовлены:

- документы и словарь проекта;
- проверки этапов 0, 1, 2, 3, 4, 5, 6 и 7;
- исходные данные сценариев;
- пассивная кинематика БВС и роя БВС;
- геометрические функции области полета;
- расчетный слой связности роя БВС и качества линии связи;
- расчетный слой звеньев роя и выбор головных БВС;
- модельная передача сообщений, очереди БВС и простая маршрутизация;
- наблюдаемый демонстратор роя, трехмерная сцена и графики показателей.

Управление, обучение и модели Simulink пока не реализованы.

## Как запустить демонстратор

Выполните файл `zapusk_demonstratora_roya.m` из корня проекта. Он
подготовит сценарий `stroi_malyi`, построит трехмерную сцену роя и
откроет окно графиков показателей качества.
