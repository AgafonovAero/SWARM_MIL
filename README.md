# Основание проекта SWARM_MIL

Проект содержит исследовательскую основу для поэтапного моделирования
роевого управления беспилотными воздушными судами в `MATLAB` и
`Simulink` версии `R2025b`.

## Где читать описание

Основное описание проекта находится в [OPISANIE.md](OPISANIE.md).

## Текущее состояние

После этапа 4 в репозитории подготовлены:

- документы и словарь проекта;
- проверки этапов 0, 1, 2, 3 и 4;
- исходные данные сценариев;
- пассивная кинематика БВС и роя БВС;
- геометрические функции области полета;
- расчетный слой связности роя БВС и качества линии связи.

Маршрутизация, звенья роя, управление, обучение и модели Simulink пока
не реализованы.
