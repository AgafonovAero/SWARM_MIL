# Основание проекта SWARM_MIL

Проект содержит исследовательскую основу для поэтапного моделирования
роевого управления беспилотными воздушными судами в `MATLAB` и
`Simulink` версии `R2025b`.

## Где читать описание

Основное описание проекта находится в [OPISANIE.md](OPISANIE.md).

## Текущее состояние

На этапе 2 в репозитории подготовлены документы, проверки и исходные
данные сценариев. Расчет движения, связи, управления и обучения пока не
реализован.
