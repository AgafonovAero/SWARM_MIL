# Журнал допущений

## Принятые допущения

- На этапе 0 проект ограничен созданием структуры, документов и проверочного запуска.
- Главный запуск проверок выполняется из файла `zapusk_proverok.m`.
- Для определения сведений о средах используются стандартные средства запроса версии расчетной и блочной среды.

## Еще не проверенные допущения

- Предполагается, что расчетная и блочная среда одной версии доступны на рабочем месте без дополнительной ручной настройки.
- Предполагается, что папка `opyty/rezultaty` доступна для записи при штатном запуске проекта.

## Ограничения применимости

- На этапе 0 отсутствуют модели движения БВС.
- На этапе 0 отсутствуют модели связи.
- На этапе 0 отсутствуют средства обучения.
- На этапе 0 не используются внешние наборы данных.
- На этапе 0 не создаются большие двоичные файлы и готовые блочные модели.
