# Описание проекта

## Назначение

Проект предназначен для поэтапного построения исследовательской основы моделирования роевого управления беспилотными воздушными судами в расчетной и блочной среде версии 2025б.

На этапе 0 создается только проверяемый каркас проекта. Способы управления, движение, связь, обучение и содержательные опыты на этом этапе не реализуются.

## Будущий состав слоев модели

- слой описания отдельного БВС;
- слой описания роя БВС;
- слой описания сети роя БВС;
- слой описания звена роя;
- слой алгоритмов управления;
- слой обучения;
- слой оценки показателей;
- слой блочного построения модели;
- слой опытов, протоколов и результатов.

## Порядок запуска проверок

1. Открыть корневую папку проекта.
2. Запустить главный файл `zapusk_proverok.m`.
3. Дождаться вывода сведений о расчетной и блочной среде.
4. Убедиться, что выведен текст `Проверки этапа 0 завершены успешно`.

## Состав этапа 0

Этап 0 включает:

- обязательные документы проекта;
- журналы допущений и изменений;
- перечень источников;
- папки для расчетных, блочных и отчетных материалов;
- единый запуск проверок.
