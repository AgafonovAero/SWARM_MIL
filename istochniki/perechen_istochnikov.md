# Перечень источников

## Нормативные источники

- Техническое задание проекта — для закрепления состава работ, этапов и порядка контроля.
- Журнал допущений проекта — для фиксации действующих ограничений и принятых упрощений.

## Научные источники

- Обзорные работы по роевому управлению БВС — для последующего выбора структуры алгоритмов и показателей.
- Работы по связности и устойчивости распределенных систем — для будущего описания сети роя и показателей качества.

## Сведения о расчетной среде

- Справочные сведения расчетной среды версии 2025б — для правил построения расчетных файлов, работы с путями и получения сведений о версии.
- Справочные сведения блочной среды версии 2025б — для будущего текстового построителя блочной модели без ручного создания двоичных файлов.
