# Описание звеньев роя

## Назначение слоя

Слой звеньев роя задает расчетное деление роя БВС на локальные группы по
уже построенному графу связности. На этапе 5 этот слой не меняет движение
БВС, а только вычисляет состав звеньев, назначает головной БВС и
рассчитывает показатели качества звеньев.

## Входные данные слоя звеньев

Для расчета звеньев используются:

- идентификаторы БВС и их расчетные состояния;
- матрица смежности графа связности;
- матрица полезности линий связи;
- матрица расстояний между БВС;
- параметры деления роя на звенья.

## Правило формирования звеньев по графу связности

Сначала в графе связности выделяются связные компоненты. Если размер
компоненты не превышает допустимый максимальный размер звена, компонента
становится отдельным звеном.

Если компонента больше допустимого максимального размера, она разбивается
жадным способом. Опорным участником выбирается БВС с наибольшей суммарной
полезностью линий внутри нераспределенного остатка. Затем к нему
добавляются связанные соседи с наибольшей полезностью линии, пока размер
звена не достигнет допустимого предела.

## Ограничение размера звена

Параметры слоя задают минимальный и максимальный размер звена. На этапе 5
минимальный размер используется как ограничение приемлемости параметров, а
максимальный размер непосредственно ограничивает процесс формирования
звеньев.

Одиночные звенья допускаются только при явном разрешении в параметрах
слоя. Если БВС не имеет связей и одиночные звенья разрешены, такой БВС
формирует собственное звено.

## Правило выбора головного БВС

Головной БВС выбирается отдельно в каждом звене по внутреннему расчетному
показателю. Показатель учитывает:

- степень БВС внутри звена;
- среднюю полезность линий связи к другим участникам звена;
- центральность по расстоянию внутри звена;
- запас энергии.

Все частные показатели нормируются в диапазон от 0 до 1. Если данных для
одного из показателей нет, используется безопасное нулевое значение и
выдается пояснение в расчетной таблице кандидатов.

## Показатели качества звеньев

На этапе 5 рассчитываются:

- число звеньев;
- число одиночных звеньев;
- средний размер звена;
- максимальный размер звена;
- средняя полезность линий внутри звеньев;
- доля БВС, назначенных в звенья;
- доля звеньев с головным БВС.

## Ограничения этапа 5

На этапе 5 не реализуются:

- маршрутизация сообщений;
- передача сообщений между БВС;
- управление движением;
- изменение траекторий БВС;
- обучение;
- модели Simulink.

Слой звеньев является расчетным и исследовательским. Он не является
системой управления реальными БВС и не задает вредного или прикладного
применения.
